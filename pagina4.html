CONCEPTOS BASICOS

  • Dominio:

    Un dominio, en el contexto de internet, es la dirección única que identifica un sitio web en la red, como "ejemplo.com" o "www.google.com".

  • Sub Dominio:

    es una extensión de tu dominio principal que te permite crear secciones o sitios web separados dentro de tu dominio principal. Es como tener una dirección web adicional que se añade a tu dominio principal.

  • URl:

    Una URL, o Localizador de Recursos Uniforme (Uniform Resource Locator), es la dirección web que identifica la ubicación de un recurso específico en la World Wide Web. Es como la dirección postal de una página web, sitio web o archivo.

  • DNS

    significa Domain Name System en inglés, es el "directorio telefónico de Internet". Traduce nombres de dominio (como "www.google.com") en direcciones IP (como "74.125.19.147"), que las computadoras utilizan para comunicarse entre sí. Sin el DNS, tendrías que recordar y utilizar direcciones IP numéricas para acceder a sitios web.

  • Hosting:

    Es un servicio que te permite publicar un sitio web o aplicación en Internet. Básicamente, es el espacio que alquilas en un servidor para almacenar todos los archivos y datos necesarios para que tu sitio web funcione, como imágenes, texto y código.

  • FTP

    FTP, que significa File Transfer Protocol o Protocolo de Transferencia de Archivos, es un protocolo de red que permite la transferencia de archivos entre sistemas en una red, especialmente Internet. Básicamente, es el conjunto de reglas que las computadoras usan para enviar y recibir archivos entre sí.

  • HTTP

    HTTP, que significa Hypertext Transfer Protocol, es el protocolo de comunicación que permite la transferencia de información en la World Wide Web. Es la base para que navegadores web y servidores se comuniquen, permitiendo la solicitud y entrega de datos como páginas web, imágenes, y otros recursos.

  • W3C

    El W3C, o World Wide Web Consortium, es una organización internacional que desarrolla estándares y directrices para la World Wide Web (WWW). Su objetivo es asegurar la evolución y el crecimiento de la web, promoviendo la interoperabilidad y accesibilidad para todos.

  • utf-8

    UTF-8 es un estándar de codificación de caracteres que utiliza una longitud variable de bytes para representar cualquier carácter Unicode. Es compatible con ASCII y se utiliza ampliamente en la web y en la comunicación electrónica.

  • HTML5 y su estructura

    El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.
